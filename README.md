# Blog Preview Card

## Overview

This is a solution to the [Blog preview card challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/blog-preview-card-ckPAB0pR2).

The challenge was to build out this blog preview card component and get it looking as close to the design as possible.

## Technologies Used

- HTML5
- CSS3

## Features

- Semantic HTML5 markup
- Custom CSS properties
- Flexbox for layout
- Responsive design

## How to Run

 Open `index.html` in your web browser.
